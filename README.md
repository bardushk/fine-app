# fine-app
Тестовое приложение проверки штрафов

##Установка Karma:
```npm install karma```

##Установка плагинов:
```npm install karma-jasmine karma-chrome-launcher```

##Создание конфигурационного файла
(на тот случай, если отсутствует файл `karma.conf.js`)

```karma init```

##Запуск тестов
```karma start```
